Python is a beautiful language just like all normal human languages. It has a very simple syntax. You must be wondering what syntax is? It is just a fancy name for the codes we write. The cool thing about python is that it is one of the most used in the world, even by big companies like google. In short, it is simple and very powerful!

A typical python code looks as simple as below:

                    
                        if 1 < 2: 
                            print("1 is less than 2") 
                        else: 
                            print("1 is not greater than 2")

    • Python as a Calculator

    Python is also a calculator and can be used to do simple and extremely complex calculations. This is very useful to do our complex task to building the biggest intelligent machines in the world. It can also help in building games too.

      Arithmetic Operations include:
    • Addition(+)
    • Subtraction(-)

    Numbers

    The python interpreter is just like the calculators we use. You can give it some instructions, and it gives the result. It also follows similar operations we use in our normal life.

    In python, we have:
  • int(integer)
  • float

    • Integer are basically whole numbers while float are fractions. You can perform all the above and more mathematical operations on these number types. '#' sign is used for commenting.

    example of addition operations:

                    
                    #integer 
                    2 + 3 = 5 
                    
                    #float 
                    4.4 + 2.5 = 6.9

    Subtraction:

                    
                    #integer 
                    5-3 = 2 
                    
                    #float 
                    8.6 - 3.4 = 5.2 
                    
                    #float and integer 
                    6.5 - 4 = 2.5

    Division:

                    
                    #integer division 
                    20 / 5 = 4 

                    #float and integer division 
                    5.5/5 = 1.1

    Multiplication:

    20 * 5 = 100

    Square root: In python. 3 2 is written as one of the below:

                    
                    #using the 'raise to power' sign 
                    3̂ 2 = 9 
                    #using double asterik sign 
                    3**2 = 9

    Lastly, modulo gives the remainder of a number. This is especially useful when we want to know if a number is even or odd. An example is shown below where it can be seen that 10 is even number because when it is divided by 2, it remains 0: 

                    
                    #Here the remainder of the the division is zero 
                    10 // 2 = 0

    numbers can also be put into an object and used later. For example, below, age_of_father, age_of_mother and age_of_child are objects that can be used later

                
                age_of_father = 27 age_of_mother = 25 age_of_child = 3
                age_of_family = age_of_father + age_of_mother + age_of_child 
                age_of_family = 55
    STRINGS

    Strings is a fancy word for alphabets. It is used almost the same way we do in real life. In python, strings are surrounded by aprostrphe(e.g "candy"). Strings can also be assigned to an object and printed. See below: 

                    
                    #assign Oyelowo to name_of_friend
                    name_of_friend = "Oyelowo" 
                    
                    #print name_of_friend
                    print(name_of_friend)

                    #this gives the result below: 
                    Oyelowo

    Strings can also be combined. This is called concatenation and it is done with the addition sign.

    Example of concatenation

                    
                    print("Face" + "book")

                    #the above gives the result below:
                    Facebook

    Strings can also be concatenated with numbers by converting the number to strings. This can be done by using "str" function. Try example below: 

                    
                    #The below convert 66 to string by using enclosing the number in "str" function
                    print("number" + str(66))

                    #result:
                    number66
